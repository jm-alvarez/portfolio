“Run Dino Run” is a offline 2D game created using Unity, this game aims to improve hand-eye coordination, to serve as stress reliever, to illustrate concepts of Computer Science and how fun and exciting programming is. It features endless running, meaning the gameplay continues until the player hits an obstacle. This provides a continuous challenge and keeps the game exciting, the game keeps track of your high score that gives you a goal to strive for and adding a competitive element to the gameplay.
